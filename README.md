## Keyboard Commands when Running Program

### Movement Controls
- 'W' - Camera Forward
- 'A' - Camera Backwards
- 'S' - Camera Left
- 'D' - Camera Right
- 'Space' - Camera Up
- 'Ctrl' - Camera Down

### UI Controls 
- 'L' - Unlock/Lock camera to mouse pointer
- 'Esc" - Exits the program

### Other Controls
- 'P' - Play a sound
- 'T' - Play bunny hop animation
